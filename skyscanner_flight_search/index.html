Decostruction Skyscanner Flight Search

Analisis

Flight Search type: Return (Round trip)

From

  • Typeahead
    • Behavior: Once a letter is introduced it appears highlighted (the matching letter) in the dropdown where the results are displayed.
      • Country (icon: flag) Country full name ( 2-letter Country code) Country full name
      • City (icon: city) City Name (“Any” when there is more than 1 airport if there is only 1 airport code) Country
      • Airport (icon: plane) Airport Full Name (IATA code) Country
  • Placeholder: Country, City or Airport
    • Preselected based in your location
  • Checkbox: Add nearby airports (under “From” selection)

To

  • Typeahead
    • Behavior: Once a letter is introduced it appears highlighted (the matching letter) in the dropdown where the results are displayed.
      • Country (icon: flag) Country full name ( 2-letter Country code) Country full name
      • City (icon: city) City Name (“Any” when there is more than 1 airport if there is only 1 airport code) Country
      • Airport (icon: plane) Airport Full Name (IATA code) Country
  • Placeholder: Country, City or Airport
    • Preselected Everywhere
  • Checkbox: Add nearby airports (under “To” selection)

Depart

  • Date picker (has an “x” to close it)
    • Specific date
      • Current month (arrows to shift to future months)
        • Days styles (weekends are highlighted in the days of the week)
          • Past days (grey disabled)
          • Available days in the month (active)
          • Available days next month (grey active)
          • Selected day (green background)
          • Hover day (blue background)
    • Whole month - Compare across a whole month (It applies to both dates pickers)
      • Cheapest month
      • Specific month (you are able to select until one year in advance)
  • Placeholder: Current date
    • Behavior
      • Preselected current day
      • Disabled to select/see past dates

Return

  • Date picker (has an “x” to close it)
    • Specific date
      • Current month (arrows to shift to future months)
        • Days styles (weekends are highlighted in the days of the week)
          • Past days (grey disabled)
          • Available days in the month (active)
          • Available days next month (grey active)
          • Selected day (green background)
          • Hover day (blue background)
    • Whole month - Compare across a whole month (It applies to both dates pickers)
      • Cheapest month
      • Specific month (you are able to select until one year in advance)
  • Placeholder: Current date
    • Behavior:
      • Preselected same date as the departure
      • If an earliest date than the departure date is selected this same date becomes the departure date.
      • Does not show or highlight the days in the between

Cabin Class and Travellers

  • DropDown Menu (has an “x” to close it)
    • Cabin Class (drop down) - Searches for whole month or destination everywhere only allows Economy
      • Economy
      • Premium Economy
      • Business Class
      • First Class
    • Adults (16+ years)
      • - # + (1 Adult mandatory)
    • Children (0-15 years)
      • - # + (Not possible to select only children)
        • Dropdown “Age of child” (As many as children's traveling)
  • Placeholder: 1 Adult, Economy

Filter

  • Checkbox: Direct flights only

Button

  • Search Flights
    • Text + icon (arrow)
      • Hover (background darker)
      • Active go to disabled state and the icon changes to a loading spinner
* All values have a preselected option that allows to trigger a search even in case that the user didn't select any value.`
